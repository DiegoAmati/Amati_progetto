### Spiegazione

1. **Flask**: Framework principale per il backend.
2. **Flask-SQLAlchemy**: Libreria per la gestione del database SQLite.
3. **Jinja2**: Motore di template utilizzato da Flask.
4. **Werkzeug**: Libreria WSGI utilizzata da Flask per la gestione delle richieste HTTP.
5. **itsdangerous**: Utilizzato da Flask per la gestione sicura dei token.
6. **click**: Libreria per la gestione delle CLI di Flask.

---

### Installazione

Per installare le dipendenze, esegui il seguente comando:

```bash
pip install -r requirements.txt
```
